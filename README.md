# Chat-Server
Chat server application using socket programming to send messages from server to client and vice versa.


## Authors

- [@ashfaaqrifath](https://www.github.com/ashfaaqrifath)


## Libraries Used
* Pyttsx3
```
  pip install pyttsx3
```

* Pathlib
```
  pip install pathlib
```

* Colorama
```
  pip install colorama
```

* Tkinter (Built-in)
```
  pip install tkinter
```
## Features

- Change narrator voices
- 8 voices
- Save audio file
- File browser dialog
